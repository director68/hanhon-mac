# 한本 for Mac

한글 키보드로 일본어를 입력하는 macOS용 입력기입니다.

## 다운로드

[Releases](https://github.com/director68/hanhon-mac/releases)에서 최신 버전을 다운로드하세요.

## 주요 기능

- **한글 → 일본어 변환**: 한글 자모를 히라가나/카타카나로 변환
- **한자 추천**: 히라가나 입력 시 한자 추천 (무료: 단일 한자, 유료: 단어 추천)
- **숫자 → 한자**: 숫자 키 입력 시 한자 숫자 추천 (一, 二, 三...)
- **단축키**:
  - `Option + K`: 카타카나 모드 전환
  - `Option + Space`: 한자 그리드 표시

## 설치 방법

1. DMG 파일 다운로드
2. DMG 열기 → `KtoJforMac.app`을 `/Library/Input Methods/` 폴더로 이동
3. 시스템 설정 → 키보드 → 입력 소스 → **한本** 추가
4. 로그아웃 후 다시 로그인

## 시스템 요구사항

- macOS 12.0 (Monterey) 이상

## 라이센스

무료 버전: 히라가나/카타카나 입력 + 단일 한자 추천
유료 버전: 단어 추천 (Viterbi 변환) 기능 포함
